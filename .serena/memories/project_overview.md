# Project Overview

## 项目名称
From Function to Transformer

## 项目类型
文档/学习项目

## 语言
- Python (脚本)
- Markdown (文档)
- LaTeX/TikZ (图表)

## 项目目的
深度学习架构学习资料，通过结构化文档和交互式笔记本解释深度学习概念。

## 技术栈
- **Python 3.10+**: 脚本仅使用标准库
- **Node.js**: mermaid-cli, markdownlint-cli2
- **LaTeX**: lualatex + dvisvgm (TikZ 图表生成)
- **Jupyter**: 可选，用于 notebooks

## 学习路线
- Part I (第1-4周): 深度学习基础
- Part II (第5-8周): 模型架构与算法
- Part III (第9-12周): 生成式模型与应用
- Part IV (第13-17周): 前沿与强化学习

## 文档语言
中文为主，代码和技术术语使用英文
