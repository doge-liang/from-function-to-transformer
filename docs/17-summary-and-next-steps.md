## 17.1 知识图谱回顾

### 17.1.1 核心概念链

```

函数思维 → 线性回归 → 神经网络
        ↓
    前向传播 → 反向传播 → 梯度下降
        ↓
    CNN → RNN → Attention → Transformer
        ↓
    预训练 → 指令微调 → RLHF → 推理增强

```

### 17.1.2 关键公式总结

| 章节 | 核心公式 |
|------|----------|
| 前向传播 | $a = \phi(Wx + b)$ |
| 梯度下降 | $W \leftarrow W - \eta \nabla L$ |
| 注意力 | $\text{softmax}(\frac{QK^T}{\sqrt{d_k}})V$ |
| Transformer | $PE_{(pos,2i)} = \sin(\frac{pos}{10000^{2i/d}})$ |
| VAE ELBO | $E[\log P(x\|z)] - D_{KL}(q(z\|x)\|P(z))$ |
| 强化学习 | $Q(s,a) \leftarrow Q(s,a) + \alpha[r + \gamma \max Q(s',a') - Q(s,a)]$ |

## 17.7 常见误区

### 17.7.1 学习误区

| 误区 | 正确做法 |
|------|----------|
| 只看视频不写代码 | 动手实现每个概念 |
| 追求最新技术 | 打好基础最重要 |
| 跳过数学推导 | 理解核心原理 |
| 盲目调参 | 系统化实验 |

### 17.7.2 实践误区

| 误区 | 正确做法 |
|------|----------|
| 过拟合训练集 | 重视验证集表现 |
| 忽视数据质量 | 花时间数据预处理 |
| 盲目复制代码 | 理解后自己实现 |
| 只看最终指标 | 关注训练过程 |
